☁ Google Sheets — Two-Way Sync

Connect this timeline to a Google Sheet. Team members can load the latest data, drag tasks to new dates, update status, add notes — then save it all back to the sheet instantly.

Name of the Google Sheet tab that contains your task list. Leave blank to use the default.
First-time setup (5 minutes):
  1. Open (or create) a Google Sheet for this project
  2. Go to Extensions → Apps Script
  3. Delete any placeholder code, paste the contents of Google_Apps_Script_Code.js
  4. Click Deploy → New deployment → Web app
  5. Set Execute as: Me  ·  Who has access: Anyone
  6. Click Deploy — copy the Web App URL
  7. Paste the URL in the field above and click Save & Connect
After setup, every team member who has the timeline URL can Load & Save to the same sheet.
